Recreation of Windows Live Messenger (MSN) using Angular & Python

By: Eric Yu

  • Launch Project
  • View Client Code on Github
  • View Server Code on Github
This project recreates the beloved chat application Windows Live Messenger (also known as MSN) using modern technologies Angular, Python, and Socket.io. The application features a main lobby chat room where users can send messages to everyone who is online. Just like the old days, the nudge and emoji libraries are integrated for added fun and expressiveness in chats. Additionally, users can engage in private one-on-one chats with friends, bringing back the intimacy of private messaging. The frontend is built with Angular, providing a sleek and responsive user interface, and an emulated environment feel of the classic Windows 7. The backend is powered by Python, leveraging the Socket.io library for real-time communication. Although user authentication is not included, this remake of Windows Live Messenger offers a nostalgic and enjoyable chat experience with modern functionalities for messaging with friends and loved ones. Relive the excitement of the past and connect with others in a whole new way with this revamped Windows Live Messenger!
